How to post

Bookmarklet

Drag and drop the bookmarklet in your bookmarks bar
click on bookmarklet for create a new bookmark
if you select a text, this is insert as comment

Post via email

From: the email used in gae.icio.us
To: post@gaeicious.appspotmail.com
Subject: the title of the post
Body: the link of the post

Feed subscription

Add your feed links in subscriptions page
gaeicious create a new bookmark for any new items in the feed
